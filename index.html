Link2Tel

Link2Tel is a simple web application that allows you to create a link to a phone call.

This application was created because Google Sheets does not support clickable phone numbers in cells.

Now, you can create a link to a phone call and paste it in your cell as:

=HYPERLINK( "https://link2tel.musicallyut.xyz/?tel=1234567890", "Call" )
